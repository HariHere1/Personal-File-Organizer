# File Organizer

A simple file organization tool currently under development.

The goal of this project is to automatically organize files in a selected directory based on their file types, such as images, documents, videos, and archives.

## Status

🚧 **Work in Progress**

The project is currently in its initial development stage. Features and documentation will be added as development progresses.

## Planned Features

* Organize files based on file type
* Create appropriate category folders automatically
* Handle unknown file types
* Basic error handling
* More features to be added during development

## Tech Stack

* Python

## Note

This project is being built step by step as a learning project. The current repository may not contain a working version yet.
